## 1. 배경 및 문제 정의

### 1.1 현재 상황
- 기존 올인원 패키지 330만원 판매 부진
- 접수 문의 및 계약 저조

### 1.2 문제 분석

| 문제 | 심각도 | 설명 |
|------|--------|------|
| **진입 장벽** | ⭐⭐⭐⭐⭐ | 330만원 선불은 중소기업에게 큰 리스크 |
| **타겟 미스매치** | ⭐⭐⭐⭐ | "올인원"이 필요한 고객이 매우 희소 |
| **가치 제안 부재** | ⭐⭐⭐⭐ | 기능 나열 중심, ROI/결과 언급 없음 |

## 2. 상품 구조

### 2.1 티어 구성 (4개 티어)

> 모든 가격은 **VAT 포함** 기준

| 티어 | 구성 | 가격 | 도메인 | Meta 포함 | 타겟 |
|------|------|------|--------|-----------|------|
| **Basic** | 랜딩 1P (4섹션, 입력폼 연동) | 30만 | ❌ | ❌ | 최저 진입점, 브랜딩 채널만 필요 |
| **Normal** | 랜딩 1P + Meta 세팅 | 60만 | ⭕ | 세팅 1회 | 광고 테스트 시작 |
| **Pro** | 홈페이지 5P + Meta 세팅 | 110만 | ⭕ | 세팅 1회 | 본격 온라인 진출 |
| **Premium** | 홈페이지 10P + Meta 세팅 | 220만 | ⭕ | 세팅 + 6개월 자동화 | 풀 구성 |

> **참고**: 호스팅은 전 티어 고객 부담

### 2.2 프로모션 (한정)

| 항목 | 내용 |
|------|------|
| 대상 | Premium |
| 정가 | 220만 (6개월 자동화 포함) |
| **프로모션가** | **165만** |
| 프로모션 혜택 | **1년 자동화 무료** (6개월 → 1년 업그레이드) |
| 기간 | ~2026년 1월 31일 |
| 한정 | **10개 기업** |

**프로모션 가치:**
- 정가: 220만 + 6개월 자동화 (16.5만 × 6 = 99만) = 319만 가치
- 프로모션: 165만 + 1년 자동화 (11만 × 12 = 132만) = 297만 가치
- **실결제: 165만원**

## 3. 역할 분담

### 3.1 POLARAD 역할
- 홈페이지/랜딩페이지 제작
- Meta 광고 세팅 및 자동화
- 입력폼 연동
- 실시간 알림 설정
- 리포팅 대시보드
- **소재 구성 코칭 및 아이디어 제공** (제작 대행 X)

### 3.2 광고주 역할
- **광고 소재 직접 준비** (이미지/영상) - 필수
- 광고비 직접 충전 (Meta 계정)
- 호스팅 비용 부담

### 3.3 핵심 메시지
```
랜딩/홈페이지 = 브랜딩 전달 채널 (도착지)
Meta 광고 = 고객 유입 엔진
광고 소재 = 성과의 핵심 (광고주 책임)
```

> **⚠️ 중요: DB 판매 서비스가 아님**
> - POLARAD는 DB를 직접 판매하지 않음
> - 고객이 **자체 DB를 생성할 수 있는 시스템**을 구축해주는 서비스
> - DB 생성 성과는 광고 소재 퀄리티와 광고비에 의존

> **주의**: 랜딩 자체는 고객 유치 목적이 아님. 
> 브랜딩 전달 채널로서 역할. 
> 실제 성과는 광고 소재 퀄리티에 의존.
> 
> **중요**: 소재 퀄리티가 낮으면 광고 효과 보장 불가.

## 4. 판매 프로세스

### 4.1 프로세스 흐름

```
1. 상담
   ↓
2. 소재 확인 (계약 전 필수)
   ↓
   → 퀄리티 OK: 계약 진행
   → 퀄리티 부족: 코칭/아이디어 제공 → 보완 후 재확인
   → 보완 불가: 진행 불가
   ↓
3. 계약 및 제작
   ↓
4. 납품 및 세팅
   ↓
5. (선택) 월정액 관리 또는 건당 유지보수
```

### 4.2 소재 정책

| 항목 | 내용 |
|------|------|
| 확인 시점 | 계약 전 |
| 기준 | 고객 반응 유도 가능한 수준 |
| 제공 | 소재 구성 코칭 + 아이디어 |
| 제외 | 제작 대행 |

## 6. 제외 사항

### 6.1 이번 기획에서 제외
- ❌ 가격 앵커링 전략
- ❌ 환불/성과보장 정책 (리스크 큼)
- ❌ 분납 옵션 (50%+50%)
- ❌ 소재 제작 서비스
- ❌ 소재 가이드/템플릿 제공
- ❌ 호스팅 포함 (고객 부담)

### 6.2 제외 사유
- 앵커링: 신뢰도 저하 우려
- 환불정책: 운영 리스크
- 소재 제작: 규모 확대, 퀄리티 이슈, 프로젝트 기간 증가
- 호스팅: 장기 비용 부담, 고객별 트래픽 차이
