비즈니스 인증 완료 후 해야 할 작업 (4개)

1 허용되는 사용 방법 설정

앱이 허용되는 사용 방법에 따라 권한, 기능 및 API를 사용하고 있는지 인증

경로: 앱 검수 → 허용되는 사용 방법으로 이동

2 데이터 처리 설정

Meta 제공 개인정보가 약관과 정책을 준수하는 방식으로 처리, 전송, 보관되는지 검증

경로: 앱 검수 → 데이터 처리로 이동

3 검수자 지침 작성

검수자가 앱에 액세스하여 기능을 검수하는 데 필요한 정보 제공

  • • 테스트 계정 정보 (이메일/비밀번호)
  • • 앱 사용법 설명 또는 동영상
  • • 주요 기능 테스트 방법

경로: 앱 검수 → 검수자 지침으로 이동

4 앱 검토 제출

모든 설정 완료 후 Meta에 앱 검토 요청 제출

경로: 앱 검수 → 제출
