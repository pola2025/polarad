# PRD: Meta 광고 V3 - Neon Glow Edition

## 1. 개요

### 1.1 목적
기존 V2 Blue 버전을 기반으로 네온글로우 효과와 섬광 애니메이션을 적용하여 신비로운 스타트업 분위기의 광고 영상 제작

### 1.2 목표
- **CTR 향상**: 시선을 사로잡는 네온 효과로 스크롤 중단 유도
- **브랜드 이미지**: 혁신적이고 미래지향적인 스타트업 이미지 구축
- **전환율 개선**: 감정적 호소력 강화로 CTA 클릭률 향상

### 2.1 컬러 팔레트

| 용도 | 색상 | Hex |

| 텍스트 | 퓨어 화이트 | `#ffffff` |

### 2.2 시각 효과

#### 네온 글로우 효과
```css
/* 기본 네온 글로우 */
text-shadow:
  0 0 10px #00f5ff,
  0 0 20px #00f5ff,
  0 0 40px #00f5ff,
  0 0 80px #00f5ff;

/* 박스 네온 글로우 */
box-shadow:
  0 0 5px #00f5ff,
  0 0 10px #00f5ff,
  0 0 20px #00f5ff,
  0 0 40px #00f5ff,
  inset 0 0 10px rgba(0, 245, 255, 0.1);
```

#### 섬광/펄스 애니메이션
- 버튼: 3초 주기 shimmer 효과
- 텍스트: 2초 주기 glow pulse
- 배경: 부드러운 gradient shift

## 3. 콘텐츠 전략 (개선된 후킹)

### 3.1 시퀀스별 메시지

#### 시퀀스 1: 후킹 (0-3초) - 감정 자극
**기존**: "광고비는 나가는데 DB는 없다?"
**개선**:
```
"매달 광고비만 100만원..."
"근데 매출은 제자리?"
💥 "지금 바로 바꿀 수 있습니다"
```
- **전략**: 구체적 금액 언급으로 공감대 형성

#### 시퀀스 2: 문제 인식 (3-6초) - 동질감
**기존**: "대행사 탓? 내 탓? 둘 다 아닙니다"
**개선**:
```
"혼자 광고 돌리다 지치셨죠?"
"대행사 맡기면 돈만 날리고..."
✨ "시스템이 답입니다"
```
- **전략**: 타겟 페인포인트 직접 언급

#### 시퀀스 3: Before (6-10초) - 현실 직시
**기존**: 현재 광고 성과 표시
**개선**:
```
"지금 당신의 광고"
광고비 100만원 → DB 24건 → 건당 42,000원
❌ "이걸로 수익이 남나요?"
```
- **전략**: 질문으로 자기 점검 유도

#### 시퀀스 4: After (10-14초) - 희망 제시
**기존**: PolaAd 사용 시 효율
**개선**:
```
"PolaAd를 쓰면"
광고비 100만원 → DB 100건 → 건당 10,000원
✅ "같은 돈, 4배 효과"
```
- **전략**: 효율 개선 효과 강조

#### 시퀀스 5: How (14-17초) - 메커니즘
**기존**: 자동화 시스템 설명
**개선**:
```
"어떻게 가능할까요?"
[광고 노출] → [AI 최적화] → [자동 DB 수집] → [실시간 알림]
⚡ "24시간 AI가 대신 일합니다"
```
- **전략**: AI 강조로 기술력 어필

#### 시퀀스 6: 증거 (17-21초) - 신뢰 구축
**기존**: 종합 분석 리포트 미리보기
**개선**:
```
"매주 받아보는 성과 리포트"
[실제 대시보드 스타일 UI]
- 이번 주 DB: 28건
- 전환율: 3.2% (업계 평균 1.8%)
- AI 추천: "목요일 예산 20% 증가 권장"
📊 "데이터가 말해줍니다"
```
- **전략**: 구체적 수치로 신뢰도 상승

#### 시퀀스 7: 가치 제안 (21-25초) - 번들 강조
**기존**: 올인원 패키지 나열
**개선**:
```
"이 모든 것이 한 번에"
✅ 홈페이지 제작
✅ Meta 광고 자동화
✅ AI 분석 리포트
✅ 인쇄물 세트
💎 "올인원 패키지"
```
- **전략**: 패키지 구성 강조, 가격은 문의로

#### 시퀀스 8: CTA (25-30초) - 긴급성
**기존**: 선착순 10개 기업, 무료 상담 신청
**개선**:
```
🔥 "할인 프로모션"
"선착순 10곳 한정"

[지금 무료 상담 받기]

polarad.co.kr
```
- **전략**: 선착순 10곳 할인 프로모션 강조

## 5. 예상 효과

### 5.1 개선 지표 (예상)

| 지표 | V2 Blue | V3 Neon (예상) | 변화 |
|------|---------|----------------|------|
| 3초 시청률 | 45% | 60%+ | +33% |
| 완료 시청률 | 12% | 18%+ | +50% |
| CTR | 1.2% | 2.0%+ | +67% |
| CPA | ₩15,000 | ₩10,000 | -33% |

### 5.2 개선 근거
1. **네온 효과**: 다크모드 환경에서 시선 집중력 극대화
2. **감정 공감**: "혼자 광고 돌리다 지치셨죠?" 타겟 페인포인트 직접 언급
3. **AI 강조**: 기술 기반 스타트업 이미지, "24시간 AI가 대신 일합니다"
4. **효과 강조**: 4배 효율 개선 시각화
5. **긴급성**: 선착순 10곳 할인 프로모션으로 행동 촉구
