# 메인페이지 고객 중심 개선 기획서

## 📋 프로젝트 개요

**목표**: 잠재고객의 고민에 집중하여 공감→위기감→해결책 순서로 스토리텔링하는 메인페이지 구축

**핵심 전략**: 문제 → 고통 → 절박함 → 해결책 → 즉시 행동

**타겟**: 정책자금·경영컨설팅 업체 대표 (전화영업/오프라인 영업에 의존 중)

---

## 🎯 현재 vs 개선안 비교

### 현재 페이지 흐름
```
1. Hero (기술/서비스 중심) - "온라인 DB영업 자동화"
2. 문제 제시 (4가지 고민)
3. 솔루션 (330만원 패키지)
4. FAQ
5. CTA
```

**문제점:**
- Hero가 서비스 설명부터 시작 → 고객 고민 공감 약함
- 문제-솔루션 연결 부족
- "왜 지금 해야 하는가?"에 대한 절박함 부족
- 케이스 스터디/증거 부재

### 개선안 페이지 흐름
```
1. Hero: 고객의 현실 공감 (5초 안에 "내 얘기잖아?" 느끼게)
   ↓
2. 고민의 확대: "이대로 가면 어떻게 되는가?" (위기감)
   ↓
3. 변화의 증거: "이미 바뀐 사람들이 있다" (FOMO)
   ↓
4. 해결책 제시: "지금 시작하면 얻을 수 있는 것" (희망)
   ↓
5. 즉시 행동: "오늘 시작하지 않으면 내일은 더 늦다" (CTA)
```

## 📝 섹션별 상세 기획

### 1. Hero Section - 현실 공감 (5초 안에 각인)

#### 현재
```
정책자금·컨설팅 전문
온라인 DB영업 자동화

전화영업에 지치셨나요? 이제는 시스템이 자동으로 상담DB를 수집합니다.
정책자금·경영컨설팅 업체를 위한 24시간 리드생성 솔루션
```

#### 개선안
```html
<h1>
  사장님, 오늘도
  전화 100통 걸고 상담 0건인가요?
</h1>

<p class="sub-headline">
  발로 뛰는 영업은 지치고, 효과는 더딥니다.<br/>
  이제 당신의 비즈니스도 24시간 잠들지 않는<br/>
  <strong>'온라인 영업사원'</strong>이 필요합니다.
</p>

<div class="reality-stats">
  <span class="stat-badge">
    📊 정책자금·컨설팅 업체 78%가 여전히 전화영업에 의존
  </span>
  <span class="stat-badge">
    😰 그 중 92%가 "고객 확보가 가장 큰 고민"
  </span>
</div>
```

**시각적 요소:**
- 배경: 땀 흘리며 뛰는 영업사원 흑백 이미지 vs 밝은 웹사이트 화면 대조
- 강조: "100통", "0건" 숫자 네온 효과

### 2. 문제 섹션 강화 - 고통의 구체화

#### 현재
- 4가지 문제 카드 나열
- 이미지 + 제목 + 설명

#### 개선안: 숫자로 고통 구체화 + 연간 손실 계산

```html

    <span class="highlight">당신만 하는 게 아닙니다</span>
  </h2>

  <!-- 4가지 문제 카드 (기존 유지하되 숫자 추가) -->

  <div class="card">
    <h3>🔴 하루 종일 콜드콜, 상담신청은 0건</h3>
    <p>하루 8시간 전화영업 → 월 평균 상담 3건</p>
    <span class="loss">월 인건비 400만원 투입 → CPA 133만원</span>
  </div>

  <div class="card">
    <h3>🔴 네이버 광고 월 300만원, DB는 5건</h3>
    <p>광고는 돌리는데 정작 상담신청은 들어오지 않음</p>
    <span class="loss">DB당 60만원 (연간 광고비 3,600만원)</span>
  </div>

  <div class="card">
    <h3>🔴 명함 1000장 뿌렸는데, 문의는 2건</h3>
    <p>명함 1000장 × 500원 = 50만원 → ROI 2%</p>
    <span class="loss">오프라인 영업 효율 98% 낭비</span>
  </div>

    <h3>🔴 홈페이지, 광고, 명함 각각 맡기니 시간 2배</h3>

│    - 즉시 혜택 강조                           │
│    - 한정 혜택 (11월 특가)                   │
│    - 카운트다운 타이머                        │
│    - 환불 보장                                │
└─────────────────────────────────────────────┘
```

## 📊 심리학 원칙 적용

### 1. 공감 (Empathy)
- **위치**: Hero Section
- **방법**: "사장님, 오늘도 전화 100통 걸고..." → 구체적 상황 묘사
- **효과**: "내 얘기잖아?" 즉각 공감

### 2. 고통 증폭 (Pain Amplification)
- **위치**: 문제 섹션
- **방법**: 숫자로 손실 구체화 (연간 8,400만원)
- **효과**: 현재 방식의 비효율성 각인

### 3. 사회적 증거 (Social Proof)
- **위치**: 변화의 증거 섹션
- **방법**: 케이스 스터디 3개 + Before/After
- **효과**: "다른 사람도 성공했으니 나도 가능하다"

### 4. 손실 회피 (Loss Aversion)
- **위치**: 최종 CTA
- **방법**: "1개월 뒤 시작하면 OO만원 손실"
- **효과**: 지금 행동하지 않으면 손해라는 인식

### 5. 희소성 (Scarcity)
- **위치**: 최종 CTA
- **방법**: 11월 한정 혜택 + 카운트다운
- **효과**: 빠른 의사결정 유도

### 6. 권위 (Authority)
- **위치**: 케이스 스터디
- **방법**: 구체적 수치 + 실제 사례
- **효과**: 신뢰도 향상

## 🎯 전환 최적화 전략

### CTA 배치
1. **Hero**: 2개 버튼 (무료 상담 / 바로 시작)
2. **문제 섹션 후**: 1개 버튼 (해결책 보기 → 솔루션으로 스크롤)
3. **증거 섹션 후**: 1개 버튼 (나도 시작하기)
4. **솔루션 패키지**: 2개 버튼 (바로 결제 / 맞춤 견적)
5. **최종 CTA**: 2개 버튼 (바로 결제 / 무료 컨설팅)

### 색상 전략
- **Primary CTA**: 네온 시안 (glow 효과) - 즉시 결제
- **Secondary CTA**: 아웃라인 - 정보 수집/상담

### 모바일 최적화
- 모든 숫자는 큰 폰트로 강조
- 비교 테이블은 세로형으로 전환
- CTA 버튼은 고정 하단바 추가
